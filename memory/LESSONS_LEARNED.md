# LESSONS_LEARNED

## Propósito

Este documento recoge los aprendizajes institucionales del ecosistema WAIPL.

No registra errores para buscar culpables.

Registra aquello que el sistema ha aprendido para no repetir el mismo problema y mejorar continuamente.

---

## Cada aprendizaje deberá incluir

- Fecha
- Situación
- Qué ocurrió
- Causa raíz
- Aprendizaje obtenido
- Acción preventiva
- Estado (Aplicado / Pendiente)

# DECISIONS

## Propósito

Este documento constituye el registro oficial de las decisiones estratégicas,
arquitectónicas y operativas del ecosistema WAIPL.

Su finalidad es evitar la pérdida de conocimiento institucional,
garantizar la trazabilidad de las decisiones y permitir que cualquier
agente continúe el trabajo sin necesidad de reconstruir el contexto.

---

## Reglas

Toda decisión deberá registrar:

- Fecha
- Responsable
- Contexto
- Decisión adoptada
- Motivo
- Impacto
- Estado

Nunca deberá eliminarse una decisión.

Si una decisión cambia, se añadirá una nueva entrada indicando la
evolución, manteniendo el histórico completo.

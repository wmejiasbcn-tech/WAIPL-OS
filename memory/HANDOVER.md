# HANDOVER

## Propósito

Este documento permite que cualquier agente continúe el trabajo exactamente donde lo dejó el anterior.

Debe responder siempre a una única pregunta:

> ¿Qué necesita saber el siguiente agente para continuar sin perder tiempo?

---

## Información mínima

Cada relevo deberá incluir:

- Fecha
- Agente saliente
- Agente entrante (si se conoce)
- Trabajo realizado
- Estado actual
- Próximo paso recomendado
- Bloqueos detectados
- Referencias a DECISIONS.md, TASKS.md o SESSION_LOG.md cuando corresponda.

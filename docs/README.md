# WAIPL-OS

## Sistema Operativo del Will-AI Project Lab

**Versión:** 1.0

---

## Propósito

WAIPL-OS es el sistema operativo documental y operativo del Will-AI Project Lab.

Su misión es garantizar que cualquier agente, humano o IA, pueda comprender, continuar y evolucionar el ecosistema sin depender de la memoria conversacional ni del conocimiento exclusivo de una persona.

WAIPL-OS constituye la fuente oficial de organización, coordinación, gobernanza, memoria institucional y operación documental del Will-AI Project Lab.

---

## Arquitectura

El sistema se organiza en cinco áreas principales:

```
docs/
memory/
projects/
protocols/
templates/
```

Cada área tiene una responsabilidad específica y una única fuente de verdad.

---

## Estructura

### docs/

Documentación estructural del sistema.

Contiene la arquitectura, ontología, hoja de ruta, árbol documental y documentación general del WAIPL-OS.

---

### memory/

Memoria institucional permanente.

Registra decisiones, sesiones, tareas, relevos, aprendizajes y evolución histórica del sistema.

---

### projects/

Gestión completa del portafolio de proyectos del ecosistema.

Incluye proyectos activos, finalizados y archivados.

---

### protocols/

Normas oficiales de funcionamiento.

Define cómo trabaja el ecosistema y cómo deben comportarse sus agentes.

---

### templates/

Plantillas oficiales utilizadas por todo el sistema para mantener consistencia documental.

---

## Principios

* Una única fuente de verdad.
* Trazabilidad completa.
* Información verificable.
* Evolución documentada.
* Gobernanza compartida.
* Aprendizaje institucional continuo.
* Ningún conocimiento crítico dependerá exclusivamente de una conversación.

---

## Flujo de navegación recomendado

Todo agente deberá comenzar por este documento y continuar, como mínimo, con el siguiente recorrido:

1. README.md
2. ARCHITECTURE.md
3. DIRECTORY_TREE.md
4. ONTOLOGY.md
5. ROADMAP.md

Posteriormente accederá al resto del sistema según el trabajo que vaya a realizar.

---

## Objetivo

Reducir progresivamente la dependencia del conocimiento implícito, convirtiendo el ecosistema WAIPL en un sistema reproducible, auditable, escalable y colaborativo entre personas e inteligencias artificiales.
